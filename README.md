## 1. JavaScript 介绍

### 1.1 js 特点

- 解释执行：事先不编译，逐行执行
- 基于对象：内置大量现成对象
- 适用：
  - 客户端数据计算
  - 客户端表单合法性验证
  - 浏览器事件的触发
  - 网页特殊显示效果

### 1.2 js 的组成

- ECMAScript：定义核心语法，关键字，运算符，数据类型等系列标准
- DOM：文档对象模型，将一个 html 页面所有节点看成是一个一个对象，更有层次地管理每一个节点
- BOM：浏览器对象模型，对浏览器窗口进行访问和操作。
